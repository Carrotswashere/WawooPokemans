# WawooPokemans

WawooPokemans is an Android application written in Kotlin that allows users to explore Pokémon by generation and view detailed information about each Pokémon. The application follows an offline-first approach, using local caching to provide a smooth experience even without an active internet connection.

---

## Features

- Browse Pokémon by generation  
- View detailed information for each Pokémon  
- Offline functionality through local data caching  
- Efficient image loading and caching  
- Modern, responsive user interface  

---

## Tech Stack

- **Kotlin** – Main programming language  
- **Jetpack Compose** – UI framework  
- **Retrofit** – REST API communication  
- **Room** – Local database for offline caching  
- **Coil** – Image loading and caching  

---

## API

The application uses data provided by the [PokéAPI](pokeapi.co).

---

## Project Architecture

- Offline-first design to reduce network dependency  
- Cached API responses stored locally with Room  
- Clear separation between data, domain, and UI layers  
- Fully built using Jetpack Compose (no XML layouts)
